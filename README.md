#####DESCRIPTION<br>
A program to track bands and the venues where they've played concerts.

#####SETUP<br>
````
$ bundle install
$ rake db:create
$ rake db:migrate
$ ruby app.rb
````

Preview available on [Heroku](https://serene-sands-1845.herokuapp.com/)
